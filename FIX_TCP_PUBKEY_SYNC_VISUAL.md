## Summary

**Problem**: Periodic pubkey sync every 5 minutes caused TCP disconnections
**Root Cause**: Excessive `interface.nodes` dict access (40-100+ operations)
**Solution**: Hash-based caching eliminates unnecessary access (0 operations)
**Result**: TCP connections remain stable, no more disconnections

**Key Innovation**: 
- Compute lightweight hash of keys (O(n) where n < 10)
- Compare hash instead of accessing interface.nodes
- Only touch interface.nodes when keys actually changed
- Automatic cache invalidation on key add/update
